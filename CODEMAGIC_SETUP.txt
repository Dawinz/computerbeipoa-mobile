Computer Beipoa Mobile — Codemagic setup
========================================

Repo:        Dawinz/computerbeipoa-mobile
Bundle ID:   com.qwantumtech.computerbeipoa
Display name: Computer Beipoa

1. Codemagic — Apple integration (required for certs + upload)
   - Team settings → Integrations → Developer Portal
   - Upload App Store Connect API .p8 key
   - Label MUST be exactly: applestoreconnectkey
   - Role: App Manager or Admin

2. Apple Developer (one-time)
   - Register App ID: com.qwantumtech.computerbeipoa
   - App Store Connect → create app with that bundle ID

3. Codemagic app
   - Add Dawinz/computerbeipoa-mobile
   - Enable "Use configuration from repository"
   - Run workflow: iOS App Store

4. Certificates (automatic)
   - Codemagic step "Set up code signing identities" + ios_signing in YAML
   - Script: xcode-project use-profiles (no manual fetch-signing-files)

5. Publishing
   - Submits to App Store review (not TestFlight)
   - Auto-releases after Apple approval

See CODEMAGIC_MOBILE_STORE_GUIDE.md for full details.
