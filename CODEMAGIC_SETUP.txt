Computer Beipoa Mobile — Codemagic setup
========================================

Repo:        Dawinz/computerbeipoa-mobile
Bundle ID:   com.qwantumtech.computerbeipoa
Display name: Computer Beipoa

1. Codemagic
   - https://codemagic.io → Add app → GitHub → Dawinz/computerbeipoa-mobile
   - Enable "Use configuration from repository"
   - Team Integrations → Developer Portal → API key label: applestoreconnectkey

2. Apple
   - Register App ID: com.qwantumtech.computerbeipoa
   - App Store Connect → New App with that bundle ID

3. Build
   - Run workflow: iOS App Store
   - Build uploads to App Store Connect and submits for **App Store review** (not TestFlight)
   - On Apple approval, release is automatic (`release_type: AFTER_APPROVAL`)

See CODEMAGIC_MOBILE_STORE_GUIDE.md for troubleshooting.
